# 深度调研

有时候我们想深入了解某个主题，大模型返回的内容可靠度往往不够。这时候可以让龙虾帮我们安装 GPT-Researcher。

---

## 一、安装 GPT-Researcher

跟之前的很多配置一样，GPT-Researcher 也可以直接让龙虾安装，静静等待它完成即可。

安装完可以让它封装成 skill，下次让它深度调研某个主题时，它就会用 GPT-Researcher 去调研了。

---

## 二、上下文频繁溢出

使用了 GPT-Researcher，龙虾在深度调研之后，经常会抛出如下错误：

```
Context limit exceeded. I've reset our conversation to start fresh - please try again.

To prevent this, increase your compaction buffer by setting agents.defaults.compaction.reserveTokensFloor to 20000 or higher in your config.
```

原因是深度调研返回的内容很大，加上平时的聊天内容，很容易超出大模型的上下文限制。

解决思路也很简单：让龙虾帮忙写一个脚本，通过 GPT-Researcher 拿到的内容不直接加入会话，而是先压缩成摘要再返回，即可解决该问题。
