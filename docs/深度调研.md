# 深度调研

有时候我们想了解一个内容，大模型返回的内容可靠度很多时候是不够的。我们可以让龙虾帮我们安装 GPT-Researcher。

---

## 一、安装 GPT-Researcher

跟之前的很多配置一样，GPT-Researcher 也可以直接让龙虾安装，我们静静等待它安装完成即可。

安装完可以让它封装成 skill，下次让它深度调研一个事情的时候，它就会用 GPT-Researcher 去调研了。

---

## 二、上下文频繁溢出

使用了 GPT-Researcher，龙虾在深度调研之后，经常会抛出如下错误：

```
Context limit exceeded. I've reset our conversation to start fresh - please try again.

To prevent this, increase your compaction buffer by setting agents.defaults.compaction.reserveTokensFloor to 20000 or higher in your config.
```

原因是深度调研返回的内容很大，加上平时的聊天内容，很容易就超出大模型的上下文限制了。

解决思路也很简单：让龙虾帮忙写一个脚本，通过 GPT-Researcher 拿到的内容不直接加入到会话中，让大模型对结果先进行压缩，之后再丢给大模型，即可解决该问题。
