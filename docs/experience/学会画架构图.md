# 让龙虾学会画架构图

有时候，我们需要龙虾帮我们画架构图。如何让龙虾学会呢？以下是我的实践，非常简单。

---

> **我**：你会画架构图吗，或者你需要哪些 skill 才会画架构图？

---

> **龙虾**：我能画的架构图类型：
>
> **1. ASCII Art（纯文本）**
>
> ```
> ┌─────────────────────────────────────────────────────┐
> │                   OpenClaw 架构                      │
> └─────────────────────────────────────────────────────┘
>                          │
>          ┌───────────────┼───────────────┐
>          ▼               ▼               ▼
>     ┌──────────┐   ┌──────────┐   ┌──────────┐
>     │  渠道层   │   │  网关层   │   │  代理层   │
>     │          │   │          │   │          │
>     │ WhatsApp │   │ Gateway  │   │  Agent   │
>     │ Slack    │──▶│ Router   │──▶│ Runtime  │
>     │ Discord  │   │ Auth     │   │ Tools    │
>     │ CLI      │   │ Model    │   │ Skills   │
>     └──────────┘   └──────────┘   └──────────┘
> ```
>
> **2. Mermaid（可渲染）**
>
> ```mermaid
> graph TB
>     A[渠道层] --> B[网关层]
>     B --> C[代理层]
>     A --> D[WhatsApp]
>     A --> E[Slack]
>     A --> F[Discord]
> ```
>
> **3. PlantUML（需要工具）**
>
> ```
> @startuml
> [渠道] --> [网关]
> [网关] --> [代理]
> @enduml
> ```
>
> **我需要的 Skill：**
>
> | Skill | 用途 | 是否已安装 |
> |-------|------|-----------|
> | Mermaid CLI | Mermaid → 图片 | ❌ 未安装 |
> | Graphviz | DOT → 图片 | ❌ 未安装 |
>
> **推荐方案：** 安装 **Graphviz + Mermaid CLI**。要我帮你安装吗？

---

> **我**：Graphviz 和 Mermaid CLI 都安装吧。

---

> **龙虾**：安装完成！
>
> | 工具 | 状态 | 版本 |
> |------|------|------|
> | Graphviz | ✅ 已安装 | 9.0.0 |
> | Mermaid CLI | ✅ 已安装 | 11.12.0 |
>
> Graphviz DOT 作为主力工具，稳定可靠，中文支持好。测试一下：

![OpenClaw 架构图](images/openclaw_layers.png)

---

> **龙虾**：**diagram skill 已创建！**
>
> 触发词：画图、架构图、流程图、diagram
>
> 下次你说"画一个架构图"，我就会直接生成了。

---

**关键原则**：发现缺 skill → 主动创建 → 持久化能力

现在，你也可以让你的龙虾学会画架构图了。 🦐
