# 永不停止的龙虾

龙虾能自主跟踪任务直到完成，无需操心。

---

## 正常情况：自动完成

给龙虾一个任务，它会：
1. 自己拆解任务
2. 自己跟踪进度
3. 一直工作到完成

你不需要干预，等它完成就行。

---

## 中断后如何恢复

以下情况会中断：
- 网络故障
- 会话超时
- 你主动停止

恢复方法：下次对话说"继续"。

龙虾会自动读取进度，从中断点继续。

---

## 进阶配置（可选）

### HEARTBEAT.md

让龙虾定期检查未完成任务，主动提醒你。

### MEMORY.md

让龙虾记住长期项目状态，跨会话持续工作。

### 模型降级链

主模型失败时自动切换备用模型，减少中断。

配置示例（`openclaw.json`）：

```json
{
  "model": {
    "primary": "tencentcodingplan/glm-5",
    "fallbacks": [
      "tencentcodingplan/hunyuan-2.0-thinking",
      "deepseek/deepseek-reasoner"
    ]
  }
}
```

---

## 核心原则

**信任龙虾**：它能自主跟踪直到完成。

**中断恢复**：说"继续"，龙虾自动恢复。

**配置可选**：HEARTBEAT、MEMORY、降级链都是锦上添花。
