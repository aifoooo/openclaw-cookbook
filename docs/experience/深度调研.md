# 深度调研

龙虾具备信息获取能力，从简单搜索到深度研究，覆盖不同场景。

---

**推荐方案**：安装 Tavily 技能。

### 安装

```bash
# 跟龙虾说："需要安装 Tavily"
# 它会自动安装，最后问你要 API Key
```

### 获取 API Key

前往 [Tavily 官网](https://tavily.com) 注册，免费额度足够日常使用。

### 使用方式

| 触发词 | 场景 |
|--------|------|
| 搜索、查一下、联网 | 快速搜索 |

## 二、深度研究

搜索适合快速获取信息，但需要深入了解某个主题时，搜索结果往往太浅、没数据、没来源。

**推荐方案**：安装 GPT-Researcher。

### 安装

```bash
# 跟龙虾说："帮我安装 GPT-Researcher"
# 它会自动完成
```

### 配置

配置文件位置：`/root/gpt-researcher/.env`

```bash
# 腾讯云 GLM-5 (OpenAI 兼容)
OPENAI_API_KEY=你的腾讯云API密钥
OPENAI_BASE_URL=https://api.lkeap.cloud.tencent.com/coding/v3

# Tavily 搜索（复用上面的 Key）
TAVILY_API_KEY=你的Tavily密钥

# LLM 配置
FAST_LLM=openai:glm-5
SMART_LLM=openai:glm-5
STRATEGIC_LLM=openai:glm-5

# Embedding - HuggingFace 本地模型
EMBEDDING=huggingface:sentence-transformers/all-MiniLM-L6-v2

# 搜索配置
RETRIEVER=tavily

# 语言
LANGUAGE=chinese
```

**注意**：GPT-Researcher 复用 Tavily 作为搜索后端，两个能力共享同一个 API Key。

### 使用方式

| 触发词 | 场景 |
|--------|------|
| 研究、调研、分析、报告 | 深度研究 |

### 报告类型

| 类型 | 说明 |
|------|------|
| `research_report` | 综合研究报告（默认） |
| `resource_report` | 资源汇总 |
| `outline_report` | 结构化大纲 |

### 输出

- **摘要**：约 3500 字，直接返回
- **完整报告**：10000+ 字，保存到 `reports/` 目录

---

## 三、上下文溢出问题

**现象**：GPT-Researcher 返回内容很大，容易超出上下文限制。

**解决方案**：完整报告先保存到文件，只返回摘要。

```
完整报告 → 保存到文件 → 压缩成摘要 → 返回摘要
```

## 四、适用场景

| 场景 | 推荐方案 |
|------|---------|
| 快速搜索答案 | Tavily |
| 深度研究/写报告 | GPT-Researcher |

| 技术选型 | GPT-Researcher |

**不适用场景**：

- 需要付费墙后的内容
- 需要专业领域深度分析
